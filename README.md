# Portfolio - Ranjani M.

This is my professional portfolio website, showcasing my work in Data Analytics, Machine Learning, and Artificial Intelligence.

## Features
- Interactive Timeline of Work Experience
- Showcase of Data Science Projects
- Comprehensive Skills Breakdown
- Staggered Animations for a Modern Feel
- Direct Email Integration

## Deployment

This site is a static HTML/CSS/JS project and can be easily hosted on **GitHub Pages**, **Netlify**, or **Vercel**.

### GitHub Pages (Recommended)
1. Initialize a git repository in this folder.
2. Create a new repository on your GitHub account.
3. Push the code to the `main` branch.
4. Go to **Settings > Pages** on GitHub and set the source to the `main` branch.

## Project Structure
- `index.html`: Main portfolio page.
- `assets/`: Folder containing images and other media.
